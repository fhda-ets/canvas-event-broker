College

College

College.js describes an object type (and really a framework) that associates a unique college specific configuration with the automation that needs to happen both in Banner and Canvas. Dispatching all Canvas operations through an instance of College ensures, among many things, that the correctly configured API client is used.

In addition, this object serves as the starting point to create derived objects through inheritance that can specify college implementations for a member function.
