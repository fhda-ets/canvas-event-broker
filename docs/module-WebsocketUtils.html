(inner) handleError(message, logger, handlerCallback, error) → {Promise}

Source:
  • WebsocketUtils.js, line 19
Utility function to handle errors that bubble up from promises chains. Ensures that the error is logged, that a predictable message format is returned to the client, and that the Promise chain emits a rejection.
Parameters:
Name Type Description
message String String describing the error condition
logger Object Logger object from calling module
handlerCallback function Completion callback for the websocket event handler
error Object The error object that was bubbled up
Returns:
A rejected Promise with the original error
